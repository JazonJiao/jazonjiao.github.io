D5P351-201022

① 10 月初把 PBS Space Time 里的视频都看了一遍，对宇宙奥秘的理解又更深了一层。10.20 ~ 10.25 又追了半遍《楚汉传奇》（第 4 遍看了），连续剧容易看了停不下来…不过也没浪费时间，剧里沟通、商量、辩论的技巧，领导力、团队文化、处事风格的对比，都对现在的我超级有用。

② 10.26，进入新学期的第 7 周，网球课时气温终于降到 25°C 以下，消耗的体力骤降一半（喝水量从一瓶下降到半瓶），要知道上周三（10.21）还是 30°C 呢。10.27，2 个多月来第 1 次测了核酸，仅 11 小时就出了结果。

③ 10.29 早上面卖力的一个做 query optimization 的组（他家是组招），主要介绍了他们的项目、问了我简历。我也根据最近学的数据库、系统设计知识，问了他们问题，比如数据库存什么、有多大，需不需要 relational algebra 知识（答曰不需要，主要是数据结构和 concurrency）。感觉良好但面完下午就拒了，无语。

④ 他们的数据库有自己的，也有用 AWS 的，主要是为其他国家提供服务时有时需要把数据放在那个国家，有时只能用第三方服务。还提到他们会看提交的 OA 代码质量，而不只是有没有过 test cases。

⑤ 话说 10.21 我心中决定去亚麻后，10.26 和 TuSimple 的 HR 随意地说，我想做一些硬核后端的任务，想和一位工程师聊聊 TuSimple 是否适合我的职业发展。让我没想到的是，她还真找到了一位搞这个方向的 Tech lead，叫 Zhou, Kai，让我 10.29 和他聊。原来 TuSimple 也有搞开发的组！

⑥ 我看了老哥的领英，UCSD 硕士毕业后做了 2 年就到 senior SWE，又过了 1 年就上管理岗了！而且方向也看上去很吸引人！我一下子有好多问题要问他…正好最近看 GSDI，了解了系统设计的一些最基本的原理。

⑦ 同时我又和 whj 讨论了一下，他和我一样录了 Amazon 和 TuSimple 但决定去亚麻了，他提到亚麻比较好解决身份问题。我倒不是很在意，我最近想了很多毕业直接回国的可能性，如果我技术过硬、申请战术得当，应该年薪 50w+ 不成问题，而且上升前景也不差。但回国还有一些其他比较重要的问题，比如选城市和落户政策，我都不是太懂；还是得早做规划。

⑧ 我现在（10.27）是这么想的：如果我和他谈的好，而且确定实习能和这位大哥做项目，那我就向 TuSimple 争取延长 ddl 到 11.15，这时就要考验我的说服能力了，我可以说：本来我想选你家了，结果亚麻给我延到了 11.15，于是我又接受了 3 家面试。

⑨ 10.29 下午和 TuSimple 的那位 TL 聊了 14 分钟，聊完以后我又推翻了去亚麻的决定。原来他家嫌 AWS 太贵，就自己建了一整个数据中心，这也太硬核了吧！TuSimple 的 infra 组包括线下和车上，他是做线下，包括建设、管理 GPU 集群。

⑩ 他说在狗家这样的大公司做 infra，由于都会用内部现成的完备系统，反而会有些局限性；他们则是多用开源的工具。但他面试大厂出来想跳槽的人，感觉大厂的好处是能了解最好的 coding 标准，包括写 code review、写 test。

⑪ 还说 TuSimple 明年有上市计划、计划 4 月实现纯无人驾驶，到时候数据量还会剧增，所以对我是好机会。不过并不能保证我能和他做，因为并不知道半年以后各组的业务需要如何。不过对这种小公司，可能各种程序不一定很正式，我选组的时候说不定可以多 argue。

⑫ 之后我找 770 和霄神（找实习积累的人脉起作用了！）询问了股票怎么算，770 说公司一般不会在熊市时上市，所以明年可能不是好时机；最好情况是我刚实习完他们上市，我也还是没太懂。

⑬ 当然，TuSimple 的实习工资算下来比亚麻少很多（我争取过涨薪，但没成功），这也让我体验到了不同职业道路带来的巨大差别。之后要到了正式 offer 信（真是小公司，这么晚才发正式 offer），ddl 推到了 11.6。

⑭ 其他影响我决定的因素有：（1）听说 TuSimple 的 new grad 面试复杂得多，两轮以后还有个 4 小时的 onsite，而亚麻申 ng 则很简单；（2）那位 TL 实习完也拿了狗家但没去，我也听说他家 intern 去向都不错；（3）亚麻听说是按简历关键词分组，还不给更新，那以我申请时的简历…估计 AWS 和 alexa 都不会要吧…？
