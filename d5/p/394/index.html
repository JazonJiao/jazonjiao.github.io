D5P394-210528

① 自从 CS142 结课，我一直在准备搭建个人网站，我明白这对我职业发展十分重要。在探索 web dev 的过程中，我似乎回到了 2 年前写 Manim.js 的时代；我发现了很多几万 stars 的 Github repo（最多的是和前端相关），沉浸在全世界码农的大 community 里，很有亲切感。

② 和 2018 年一样，我开始无聊的时候就看 Coding Train 的视频学习 web dev，如果世界上所有的编程教程都是他来做，那就好了！还看了其他人的在线编程视频，感觉比看教程文章高效一些。另外，Derek Banas 怎么什么都会，吾辈楷模啊！

③ 5.28 晚上，我学了些如何线上部署一个 project、各家服务器特点的比较，有种醍醐灌顶的感觉！CS145、142、110 学了那么久开发岗技术，我还从来没实践过代码上线这一步呢。

④ 至 6.2，我已经调研了很久，却还没正式开始做网站，因为有些设计上的决定是不可逆的，必须在开始前做充分的了解，比如前端用 Bootstrap 还是 Material-UI，选了哪个，要出了问题，感觉换别的会很困难；这应该是软件工程的一个大规律。如果真是在工作里，会有机会给我这样光研究不做吗…想到了 SCGame 里有专门的 researcher 岗位，解锁新工具都需要积累足够的 research points…

⑤ 为了用 markdown 写博客，我试了 Hexo，感觉自定义的自由度比较小，遂尝试 jekyll，6.3 晚上花了 2 小时装 ruby on rails，但死活成功不了，哎。（6.11 继续尝试，还是无果，想起了 2018 年底尝试装 manim 的场景…）还是直接把 markdown 转化成 html 吧，这其实是可以用 regex 解决的事情。但发现 JavaScript 无法直接读取文件，于是又回到了起点…

⑥ 6.3 收到了上周申的 NLP 项目的面试，由于需要 web dev 技能，我就狂肝了近 10 小时，搭好了个人网站的基本框架，次日下午面试时给学长展示了一下。这个项目是关于搭建 QA 系统的，主要任务是 crowdsource 搭数据集，要写一些前后端的代码。（6.12 得知他们最后选了我的 224n 队友 966，这也太巧了！哎，毕竟我只有秋季有空，966 暑假就能开始做了。）

⑦ 6.5 正式放暑假，我给网站设计了配色，蓝色为主色调。Kevin Powell 一个视频里说网站的色调就是黑、白加另一个颜色；除非能协调得很好，一般不会用 2+ 个自定义色调。

⑧ 我还发现了两个极其有创意的个人网站，分别以方向盘可控的小车、马里奥风格的游戏的方式，呈现作者的简介、作品、技能。是啊，如何展现我的过往经历，是个让我很头疼的问题。对混迹学界的朋友们，很简单，就是介绍自己的研究兴趣、发表历史；而我应该侧重给 employer 展现我的技能，这要怎么做呢。这里的原则是用 STAR，先介绍背景引入，再讲自己做了什么。

⑨ 期末的 2 周闲得蛋疼，二刷了《人民的名义》，太上头了！每次省委常委会都好精彩，让我更好体会到了神仙打架、暗流涌动是怎样的场面。对我十分有用的是，我了解了中国政府这个庞大组织的结构、职级划分、上升通道。从结构上，整个政府机构就是个放大版的班委，具体职责的分配，我小学中学已经有比较深的体会了。晋升的话靠上级推荐、常委会开会讨论表决。

⑩ 剧中不少高官的子女都在美国读书，不过让我疑惑的是，我在 Vandy 和 Stanford 还从没遇到过；我和我身边同学的父母基本都是挺普通的人，同学们也大都是爱国、想念祖国的。000 判断来美国的有钱的多于有权的，他在加拿大倒是遇到过高官子女。（side note：780 说 CS161 的 TA 里有一位墨西哥女足国家队成员，Teresa Noyola，我们系真是人才辈出啊。）

⑪ 6.7 和 458 的好友 000 面了个基，000 是 KH scholar，这个项目每年招约 60 人，是和研究生项目一起申请的；提供 3 年的奖学金，包学费、每年给 1.3w 生活补助，妈呀，真是太壕了？！000 介绍了她的本科、硕士论文，我们探讨了一些科技与社会、大国博弈的关系的话题（如华为），她从文科生的角度提供了很不错的见解。

⑫ 6.9，8 个人组队去六旗坐过山车，刚进园时 782 看着游乐园地图，说要用 A *算法找到去吃饭的最短路径，笑死。

⑬ 我还是第一次坐带翻滚的轨道，太上头了！就是侧向的旋转让人感觉有点晕。还有个项目是在圆环里转圈，有一共近 10 秒的静止倒挂，倒挂时我对面的 763 还很开心地伸手比划着耶，orz…下来之后我感觉反胃，恶心的感觉在结束后 30 分钟达到顶峰，让我回想起了 2018 下半年的各种糟糕经历…1 小时后基本消退。

⑭ 我了解到 782 研一只上了 4 节课，都是 AI，和我没有重合；他在 SVL 做 robotics，但没有真的机器人，而是做 simulation。786 搞过从 simulation 到现实 robots 的 transfer learning。000 评价 robotics 有科研的前景，但落地难。783 在 Percy 的 lab 写 web，非常爽。
